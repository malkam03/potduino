potduino module¶

class potduino.Potduino(scl_pin: int = 5, sda_pin: int = 4, ds_pin: int = 14, sleep_minutes: int = 60)¶

Bases: object

The potduino class encapsulates the hardware functionality

Parameters

  • scl_pin (int) – I2C Clock pin in which the ATH10 and BH1750 are connected to, default 5.

  • sda_pin (int) – I2C Data pin in which the ATH10 and BH1750 are connected to, default 4.

  • ds_pin (int) – OneWire Data pin in which the BH1750 sensors are connected to, default 14.

  • sleep_minutes (int) – minutes in which the device will be in deep sleep between operations.
